Machine Learning

Prédiction de Churn Client

Python, Scikit-learn, Pandas | Durée: 4 semaines

Contexte du Projet

Dans le secteur bancaire, la fidélisation des clients est cruciale. Ce projet visait à développer un modèle prédictif capable d'identifier les clients à risque de départ (churn) afin de leur proposer des offres ciblées.


Méthodologie

  • Collecte et nettoyage des données clients (historique, transactions, démographie).
  • Analyse exploratoire des données (EDA) pour identifier les corrélations.
  • Feature Engineering et sélection des variables pertinentes.
  • Entraînement de modèles (Random Forest, XGBoost, Logistic Regression).

Résultats

Le modèle final (XGBoost) a atteint une précision de 92%. Il a permis d'identifier les principaux facteurs d'attrition, tels que l'inactivité récente et le nombre de réclamations.
