Deep Learning

Classification d'Images E-commerce

TensorFlow, Keras, Python | Durée: 3 semaines

Contexte du Projet

Automatisation du processus de catégorisation des produits sur une marketplace en utilisant leurs images, pour réduire le travail manuel et les erreurs.


Méthodologie

  • Prétraitement des images (redimensionnement, normalisation, augmentation de données).
  • Construction d'un réseau de neurones convolutif (CNN).
  • Transfer Learning à l'aide de modèles pré-entraînés (VGG16, ResNet).
  • Optimisation des hyperparamètres.

Résultats

Le modèle classe correctement les produits dans 15 catégories différentes avec une accuracy de 89%. Déploiement du modèle via une API Flask.
